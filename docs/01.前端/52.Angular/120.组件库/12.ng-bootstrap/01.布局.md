## 循环3分行示例

```html
<style>
  .col-sm-4{
    border: solid 1px red;
  }
</style>
<div class="container" >
  <div class="row"  >
    <div class="col-sm-4"*ngFor="let i of list;let key=index">
      {{i}}
    </div>
  </div>
</div>


```

data

```typescript
list:any[]=['one','two',5,7,8,56,89,56,23,12,45,78,56,23,45,89,56,12,36,56]
```
