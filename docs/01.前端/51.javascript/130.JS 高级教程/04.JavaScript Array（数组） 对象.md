数组对象的作用是：使用单独的变量名来存储一系列的值。

## 数组方法和属性

使用数组对象预定义属性和方法：

```js
var x=myCars.length             // myCars 中元素的数量
var y=myCars.indexOf("Volvo")   // "Volvo" 值的索引值
```
