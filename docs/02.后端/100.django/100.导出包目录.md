导出Django项目所安装的包和快速安装Django项目所需要的包

## 导出项目所安装的包：

命令：pip freeze > requirements.txt
注意：requirements.txt的内容是项目所安装的包；

## 快速安装Django项目所需要的包：

命令：pip install -r requirements.txt
